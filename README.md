Pokédex

A responsive Pokédex web application built with vanilla JavaScript and the PokéAPI. Browse Pokémon, search by name, explore detailed information, and navigate seamlessly through an interactive dialog.

Project Description

This project is a fully responsive Pokédex that retrieves Pokémon data from the PokéAPI.

Users can browse Pokémon cards, load additional Pokémon, search by name, and open a detailed dialog containing information about each Pokémon.

The project was developed with a strong focus on clean code, modular architecture, responsive design, and reusable components.

Features
Display Pokémon cards
Load More functionality
Search Pokémon by name
Live search with multiple results
Responsive design
Pokémon detail dialog
About tab
Base Stats with progress bars
Evolution chain
Previous / Next navigation
Keyboard navigation (Arrow Left / Arrow Right)
Close dialog with ESC
Close dialog by clicking outside
Dynamic card colors based on Pokémon types
Official Pokémon artwork
Smooth dialog animations
